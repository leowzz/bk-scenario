# 仪表盘与图形编辑器规范

## 页面信息架构

- 顶部：项目切换器、全局搜索、执行入口。
- 左侧：规则列表（可切换“大按钮视图”）。
- 中间：规则图编辑画布。
- 右侧：节点配置、变量面板、执行详情。

## 大按钮模式

- 每个规则在项目主页渲染为一个大按钮卡片。
- 点击按钮直接触发默认执行（可带参数弹窗）。
- 按钮显示最近一次执行状态与耗时。

## 图模式

- 支持新增 `sql/python/shell/log/store/load` 节点。
- 支持连线、拖拽、节点配置编辑。
- 提供 `保存图` 与 `运行` 操作。

## 配置体验

- SQL 节点需要选择 MySQL 连接器。
- Redis 节点（load/store 扩展）可选择 Redis 连接器。
- Python/Shell 节点在 UI 显示超时与安全策略提示。

## 可观测性

- 执行列表支持按状态、时间、规则筛选。
- 执行详情展示每个步骤的输入摘要、输出摘要与错误。
- `stored_data` 可在执行详情中直接查看。
